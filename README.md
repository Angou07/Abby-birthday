# Birthday Greeting Website

## Description
This project is a birthday greeting website that allows users to send customized greetings to their friends and family. It provides an interactive interface where users can select different greeting templates, enter personalized messages, and choose images to accompany their greetings.

## Features
- Variety of greeting templates
- Customizable messages
- Image selection for a personalized touch
- User-friendly interface
- Option to share greetings via social media

## Technologies Used
- HTML
- CSS
- JavaScript
- [Any relevant libraries or frameworks]

## Usage
To use this project:
1. Clone the repository: `git clone https://github.com/Angou07/Abby-birthday.git`
2. Open the `index.html` file in your browser.
3. Follow the on-screen instructions to send a birthday greeting.
